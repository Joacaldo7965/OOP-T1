## Compilación y ejecución
Primero se accede a la sección correspondiente:
```sh
cd stage1/
```
Para sólo compilar la sección:
```sh
make
```
Para compilar y ejecutar la sección:
```sh
make run
```
Para limpiar los archivos:
```sh
make clean
```
